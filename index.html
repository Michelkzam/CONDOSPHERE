📖 Como rodar este código no seu Supabase (Passo a Passo para Leigos)
  1. Passo 1: Acesse o painel do seu projeto no site oficial do Supabase.
  2. Passo 2: No menu lateral esquerdo do Supabase, clique no ícone **SQL Editor** (representado pelo símbolo de terminal >_).
  3. Passo 3: Clique no botão verde "New Query" (Nova Consulta) no topo da tela do editor.
  4. Passo 4: Clique no botão de atalho "📋 Copiar SQL" (localizado acima deste painel) para copiar todo o script de tabelas.
  5. Passo 5: Cole o código copiado na área de texto preta do editor do Supabase e clique no botão verde "Run" (Executar) no canto inferior direito.

✓ Pronto! Suas tabelas e chaves de segurança serão criadas instantaneamente na nuvem do Supabase!
